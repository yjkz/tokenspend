# i18n 多语言支持设计文档

## Context

Token Dashboard 当前所有 UI 文字为硬编码英文。需要增加中文版，支持语言切换，默认中文。

## 方案

自建 React Context + JSON 翻译文件，无外部依赖。

## 架构

```
src/i18n/
  index.ts              # LanguageContext + useLanguage hook + t() 函数
  locales/
    en.ts               # 英文翻译 (35 key)
    zh.ts               # 中文翻译 (35 key, 默认)
```

## 核心实现

### LanguageContext (`src/i18n/index.ts`)

- `LanguageProvider` 包裹整个 App
- 状态 `lang: 'zh' | 'en'`，初始值从 `localStorage.getItem('lang')` 读取，默认 `'zh'`
- `setLang` 切换语言并写入 localStorage
- `t(key, params?)` 函数：根据当前 lang 返回翻译文本，支持 `{ count: 39 }` 插值

### 侧边栏语言切换 (`AppLayout.tsx`)

在导航链接下方增加语言切换区域：

```
Token 仪表盘
━━━━━━━━━━
仪表盘
会话
━━━━━━━━━━
🌐 中文 | English
```

中文/英文按钮样式：选中态高亮，未选中态灰色。

## 默认语言

中文 (`zh`)。首次访问显示中文，用户可切换。

## 验证

1. `node start.cjs` 启动
2. 默认显示中文界面
3. 点击 "English" 切换为英文
4. 刷新页面，语言保持英文（localStorage）
5. 所有页面文字正确切换
